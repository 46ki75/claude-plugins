# rust

A Claude Code plugin bundling Rust development skills. Each skill activates on
demand based on its `description`.

## Bundled skills

| Skill | Covers |
| --- | --- |
| `rust-toasty` | Expert guidance for the Toasty Rust ORM (v0.7) |

## Install

```bash
/plugin marketplace add 46ki75/claude-plugins
/plugin install rust@46ki75-plugins
```
