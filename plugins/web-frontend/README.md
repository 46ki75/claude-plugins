# web-frontend

A Claude Code plugin bundling frontend and browser-extension development skills.
Each skill activates on demand based on its `description`.

## Bundled skills

| Skill | Covers |
| --- | --- |
| `qwik` | Qwik (v1/v2) and its meta-framework (Qwik City / Qwik Router) |
| `web-view-transition` | The View Transition API (same-document and cross-document) |
| `wxt` | Building browser extensions with WXT |

## Install

```bash
/plugin marketplace add 46ki75/claude-plugins
/plugin install web-frontend@46ki75-plugins
```
