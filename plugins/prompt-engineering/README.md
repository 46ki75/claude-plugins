# prompt-engineering

A Claude Code plugin bundling eval-driven prompt refinement skills. Each skill
activates on demand based on its `description`.

## Bundled skills

| Skill | Target |
| --- | --- |
| `prompt-evaluation` | Anthropic / Claude API prompts |
| `prompt-evaluation-claude-code` | Prompt refinement run entirely inside Claude Code |

## Install

```bash
/plugin marketplace add 46ki75/claude-plugins
/plugin install prompt-engineering@46ki75-plugins
```
