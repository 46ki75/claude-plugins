# authoring

A Claude Code plugin bundling document-authoring skills. Each skill activates on
demand based on its `description`.

## Bundled skills

| Skill | Covers |
| --- | --- |
| `markdown` | Markdown linting and automated fixing via markdownlint-cli2 |
| `mermaid` | Creating diagrams and visualizations with Mermaid syntax |

## Install

```bash
/plugin marketplace add 46ki75/claude-plugins
/plugin install authoring@46ki75-plugins
```
